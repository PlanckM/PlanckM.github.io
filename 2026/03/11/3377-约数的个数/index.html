3377. 约数的个数

一、完整带注释代码

二、代码实现思想

1. 核心需求

输入n个测试用例（n个整数x），依次统计每个整数x的正因数个数，并输出每个x对应的因数个数；正因数指能整除x且大于0的整数（如x=6，因数为1、2、3、6，个数为4）。

2. 核心逻辑（因数统计技巧）

核心优化点：利用“因数成对出现”的特性，减少循环遍历范围，提升程序效率，具体逻辑如下：

  • 对于任意正整数x，若j是x的因数，则x/j也一定是x的因数（成对出现），例如x=12，j=2是因数，x/j=6也是因数；
  • 成对的因数中，最大的“较小因数”不会超过sqrt(x)（平方根），例如x=16，sqrt(x)=4，成对因数为(1,16)、(2,8)、(4,4)，遍历到4即可覆盖所有因数；
  • 特殊情况：当x是完全平方数时（如4、9、16），会出现j = x/j的情况（如x=4，j=2，x/j=2），此时需避免重复统计，只计数1次。

3. 主函数核心流程（分步骤）

  1. 输入准备：读取测试用例个数n，确定需要处理的整数个数；
  2. 循环处理每个测试用例： 读取当前需要统计因数的整数x；
  3. 初始化累加变量sum为0（用于记录x的因数个数）；
  4. 遍历判断：从j=1循环到j≤sqrt(x)，对每个j判断是否是x的因数；
  5. 因数统计：若j是x的因数，sum加1（统计j）；若成对的因数k=x/j与j不相等，sum再加1（统计k）；
  6. 结果输出：每个测试用例处理完毕后，输出sum（即x的因数个数）；
  7. 程序结束：n个测试用例全部处理完毕，主函数正常返回，程序终止。

4. 关键细节说明

  • 头文件作用：引入math.h是为了使用sqrt()函数，用于计算x的平方根，确定循环的终止条件，避免无效遍历；
  • 循环范围：j从1开始（因数是正整数，不包含0），到sqrt(x)结束，相较于遍历1到x，大幅减少循环次数（如x=10000，遍历到100即可，而非10000）；
  • 去重逻辑：判断k != j是核心，避免完全平方数的因数重复统计（如x=9，j=3，k=3，只统计1次，确保因数个数正确）；
  • 边界处理：x=0不纳入考虑（题目隐含输入为正整数），x=1时，sqrt(1)=1，j=1是因数，k=1，只统计1次，输出sum=1（符合1的因数只有1的规律）。

5. 示例说明（帮助理解）

示例1：输入n=1，x=6；sqrt(6)≈2.45，循环j=1、2；

j=1：6%1==0 → sum=1；k=6/1=6≠1 → sum=2；

j=2：6%2==0 → sum=3；k=6/2=3≠2 → sum=4；

输出4（6的因数：1、2、3、6，共4个）。

示例2：输入n=1，x=4；sqrt(4)=2，循环j=1、2；

j=1：4%1==0 → sum=1；k=4≠1 → sum=2；

j=2：4%2==0 → sum=3；k=2==2 → 不加分；

输出3（4的因数：1、2、4，共3个）。
