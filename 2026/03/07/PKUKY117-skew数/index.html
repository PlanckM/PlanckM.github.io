PKUKY117 skew数

一、带完整注释的代码

二、代码实现思想

本代码的核心功能是：将输入的Skew数（由0和1组成的字符串）转换为对应的十进制数，支持多组输入，核心是理解Skew数的编码规则并通过编程实现转换逻辑。

1. Skew数编码规则（核心前提）

Skew数是一种特殊的二进制衍生编码，其每一位的权重与普通二进制不同，具体规则如下：

  • 设Skew数的各位（从右往左，即低位到高位）依次为第1位、第2位、…、第n位；
  • 第k位（k≥1）的权重为 （而非普通二进制的）；
  • Skew数的十进制值 = 每一位的数字（0或1） × 对应位的权重，所有位的贡献值累加求和。

示例：Skew数“101”（原字符串），反转后为“101”（索引0、1、2对应原数的第1、2、3位）：

十进制值 = 1×(2¹-1) + 0×(2²-1) + 1×(2³-1) = 1×1 + 0×3 + 1×7 = 8。

2. 代码实现步骤（核心逻辑）

  1. 准备工作：引入所需库（输入输出、算法库），定义power函数用于计算权重的基础（2的n次方）；
  2. 读取输入：使用while循环实现多组输入，每次读取一个Skew数字符串；
  3. 字符串反转：调用reverse函数反转输入字符串，目的是将原Skew数的低位（右侧）转到字符串左侧，方便通过索引i直接对应“第i+1位”，简化权重计算；
  4. 计算Skew数：遍历反转后的字符串，对每一位执行两个操作——计算当前位的权重（）、将字符转为数字，然后累加每一位的贡献值；
  5. 输出结果：每次计算完成后，输出当前Skew数对应的十进制值，进入下一组输入循环。

3. 关键技术点解析

  • reverse函数：来自algorithm库，用于反转字符串，语法为reverse(字符串.begin(), 字符串.end())，核心作用是简化位索引与权重的对应关系；
  • power函数：自定义函数，用于计算2的e次方，为权重计算（）提供基础，循环实现简单易懂；
  • 字符转数字：通过“in[i] - ‘0’”实现，利用ASCII码差值（字符’0’的ASCII码为48，’1’为49），快速将字符形式的0/1转为int类型的0/1；
  • 多组输入：while (cin >> in) 循环，只要有输入就会持续执行，直到用户手动结束输入（如Ctrl+Z）。

4. 注意事项

  • 宏定义语法：把int 当long long用
  • 输入要求：输入的字符串必须仅包含0和1，若包含其他字符，会导致“in[i] - ‘0’”结果异常，进而计算错误；
  • 数据范围：power函数返回int类型，当Skew数长度过长（如超过30位），2的30次方会超出int范围，可将power函数和skew变量改为long long类型避免溢出。
