SJUKY122 字母统计

一、带完整注释的代码

二、代码思想（核心逻辑）

本代码的核心功能是：统计用户输入字符串中所有大写英文字母（A-Z）的出现次数，并按A-Z的顺序输出每个字母及其对应次数，即使字母未出现（次数为0）也会正常输出，确保统计结果的完整性。

1. 整体思路框架

  1. 准备工作：引入所需库（输入输出、map容器、字符串），定义相关变量（存储输入的字符串、存储字母-次数的map容器）；
  2. 初始化map容器：将A-Z每个大写字母作为键，初始次数设为0，避免后续统计时遗漏未出现的字母；
  3. 统计字母出现次数：遍历用户输入的字符串，筛选出大写字母，对对应字母的次数进行累加；
  4. 输出结果：通过迭代器遍历map容器（默认A-Z顺序），逐个输出每个字母及其出现次数。

2. 关键技术点解析

  • map容器的使用：map 适合存储“键-值”对应关系，此处键为大写字母（char类型），值为出现次数（int类型），且map会自动按键的ASCII码升序排序，无需额外处理排序逻辑；
  • 迭代器的使用：auto pair = mp.begin() 定义map迭代器，通过迭代器遍历map中的所有键值对，pair->first 表示当前键（字母），pair->second 表示当前值（次数）；
  • 字符筛选：通过if(i >= ‘A’ && i <= ‘Z’) 判断字符是否为大写字母，避免统计小写字母、数字、符号等无关字符，确保统计准确性；
  • 初始化操作：提前将A-Z所有字母的次数设为0，保证即使某个字母未在输入中出现，也能输出“字母:0”，让结果更完整。

3. 输入输出示例

输入：ABCABZ 输出： A:2 B:2 C:1 D:0 …（省略中间字母） Y:0 Z:1

4. 注意事项

  • 输入字符串中若包含小写字母、数字、符号等，会被自动忽略，仅统计大写字母；
  • map容器默认按键的升序排列（A-Z），输出顺序固定为A到Z；
  • auto关键字用于自动推导迭代器类型，简化代码书写，C++11及以上版本支持。
